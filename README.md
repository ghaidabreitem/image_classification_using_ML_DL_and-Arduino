# image_classification_using_ML_DL_and-Arduino
🧠 Project Overview

This project focuses on image classification using machine learning and deep learning models integrated with a real-time camera system and Arduino-based hardware response.

The system is designed to classify images of cats and dogs captured through a camera. Based on the prediction result:

🐱 If a cat is detected → a green LED turns on
🐶 If a dog is detected → a red LED turns on

This creates a simple real-time AI-based system combining camera input, machine learning models, and embedded hardware (Arduino).

📊 Dataset

The models were trained on a balanced dataset containing:

12,499 images of cats
12,499 images of dogs
🤖 Models Used

Multiple models were implemented and compared:

CNN
KNN
SVM
Logistic Regression
📈 Evaluation Metrics

The models were evaluated using:

Accuracy
Precision
Recall
F1-score
📊 Results
CNN: 96% (Best performance)
KNN: 60%
SVM: 57%
Logistic Regression: 56%
📌 Conclusion

The results show that CNN significantly outperformed traditional machine learning models due to its ability to learn complex patterns directly from image data.

This confirms that deep learning is more effective for image classification tasks compared to classical machine learning methods, especially with large datasets.

🏁 Summary

This project demonstrates an end-to-end system that combines:
Machine Learning + Deep Learning + Camera-based input + Arduino hardware response
